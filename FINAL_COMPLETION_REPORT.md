## 💡 技术创新点

### 1. 双层系统架构
```javascript
// 表世界/里世界通过CSS opacity和pointer-events实现
// 完全隔离，无状态冲突
body.tomori-mode {
    #canvas-container { opacity: 0.3; pointer-events: none; }
    #ui-layer { opacity: 0.3; pointer-events: none; }
}
```

### 2. MediaPipe手势识别
```javascript
// V手势检测的精妙算法
const isVGesture = (landmarks) => {
    const indexMiddleDistance = distance(index, middle) > 0.05
    const middleRingDistance = distance(middle, ring) < 0.03
    return indexMiddleDistance && middleRingDistance
}
```

### 3. Canvas粒子系统
```javascript
// 高性能粒子动画，支持desktop(7500)/mobile(4000)
const particle = {
    x, y,
    vx: random(-0.5, 0.5),
    vy: random(-0.5, 0.5),
    opacity: random(0.3, 1),
    update: () => { x += vx; y += vy; }
}
```

### 4. 状态管理
```javascript
// 6个主要状态 + Tomori模式
states: [INTRO, IDLE, COUNTDOWN, CAKE, BLOWING, CELEBRATION]
+ tomoriMode: false/true
+ tomoriCurrentStage: 1/2
// 确保转移的线性和稳定性
```
