## 🎯 项目简介

一个沉浸式的数字艺术生日贺卡，通过两幕式的场景设计、手势识别、星空动画和互动故事，诠释"观测者"对被观测者的爱与陪伴。

**核心概念**：模仿友利奈绪手中的摄像机，将观看方比作一个冷静、持续、忠实的观察者——在闪光时刻记录美好，在低谷时期成为灯塔。

## 🛠️ 技术栈

```
前端框架：HTML5 + CSS3 + JavaScript ES6+
3D渲染：Three.js r128 (WebGL)
2D渲染：Canvas 2D API
手势识别：MediaPipe Hands 0.4.1646
UI框架：Tailwind CSS
音频：Web Audio API
动画：CSS3 Transitions + requestAnimationFrame
```
